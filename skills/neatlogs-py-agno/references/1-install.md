# Step 1: Install neatlogs (latest)

1. Call `detect_package_manager`.
2. Install LATEST neatlogs + agno (+ the provider, e.g. openai):
   - pip: `pip install --upgrade neatlogs agno openai`
   - uv: `uv add --upgrade neatlogs agno openai`
   - poetry: `poetry add neatlogs@latest agno openai`
3. Background install; continue.
