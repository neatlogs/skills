# Step 1: Install neatlogs (latest)

1. Call `detect_package_manager`.
2. Install the LATEST neatlogs + dspy (always newest; upgrade flag):
   - pip: `pip install --upgrade neatlogs dspy-ai`
   - uv: `uv add --upgrade neatlogs dspy-ai`
   - poetry: `poetry add neatlogs@latest dspy-ai`
3. Background install; continue immediately.
