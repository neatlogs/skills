# Step 1: Install neatlogs (latest)

1. Call `detect_package_manager`.
2. Install LATEST neatlogs + strands-agents:
   - pip: `pip install --upgrade neatlogs strands-agents`
   - uv: `uv add --upgrade neatlogs strands-agents`
   - poetry: `poetry add neatlogs@latest strands-agents`
3. Background install; continue.
