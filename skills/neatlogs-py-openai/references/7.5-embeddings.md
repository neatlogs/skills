# Step 7.5: Embeddings — Decorate ONLY Custom Ones

## The Rule

| Embedding source | Action |
|---|---|
| Wrapped-client call (`client.embeddings.create()` on a `neatlogs.wrap()`'d OpenAI client, or a Path-B provider in `instrumentations=[]`) | **Do NOTHING** — auto-traced as an EMBEDDING span |
| Custom embedder (local model, `sentence-transformers`, raw HTTP, hand-rolled function) | Add `@neatlogs.span(kind="EMBEDDING")` + attributes |

## Why wrapped/instrumented embeddings need no decoration

`neatlogs.wrap()` patches the client's `embeddings` resource (and the Path-B instrumentor patches the provider SDK), so `client.embeddings.create()` is captured as an EMBEDDING span automatically — with model, token count, and vector data.

```python
# ❌ WRONG — client.embeddings.create() on a wrapped client is already traced. Duplicate EMBEDDING span.
@neatlogs.span(kind="EMBEDDING")              # REMOVE
def embed(texts):
    return client.embeddings.create(model="text-embedding-3-small", input=texts)

# ✅ RIGHT — leave it alone. The wrapped client captures it.
def embed(texts):
    return client.embeddings.create(model="text-embedding-3-small", input=texts)
```

## When it IS custom — decorate with attributes

Only when the embedding is produced WITHOUT a wrapped/instrumented provider SDK — e.g. a local `sentence-transformers` model, a custom REST endpoint, or a hand-written function:

```python
import neatlogs

def embed_texts(texts: list[str]) -> list[list[float]]:
    with neatlogs.trace("embed_texts", kind="EMBEDDING") as span:
        span.set_attribute("neatlogs.embedding.model_name", "all-MiniLM-L6-v2")
        span.set_attribute("neatlogs.embedding.text", texts[0] if texts else "")
        vectors = local_model.encode(texts)
        span.set_attribute("neatlogs.embedding.token_count", sum(len(t.split()) for t in texts))
        return vectors.tolist()
```

Or the decorator form for a dedicated function:

```python
@neatlogs.span(kind="EMBEDDING")
def embed_texts(texts: list[str]) -> list[list[float]]:
    return local_model.encode(texts).tolist()
```

## Recognized embedding attributes

Set any of these on the span when known (all optional):

| Attribute | Meaning |
|---|---|
| `neatlogs.embedding.model_name` | The embedding model identifier |
| `neatlogs.embedding.text` | The input text being embedded |
| `neatlogs.embedding.token_count` | Tokens consumed |
| `neatlogs.embedding.vector` | The resulting vector (rarely set — large) |

## Verification

- [ ] No `client.embeddings.create()` call has `@neatlogs.span(kind="EMBEDDING")` or a trace wrapper.
- [ ] Custom embedders (local model / raw HTTP) — IF present — have `@span(kind="EMBEDDING")` or `trace(kind="EMBEDDING")` with at least `neatlogs.embedding.model_name`.
