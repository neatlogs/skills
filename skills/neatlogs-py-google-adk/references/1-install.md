# Step 1: Install neatlogs (latest)

1. Call `detect_package_manager`.
2. Install LATEST neatlogs + google-adk:
   - pip: `pip install --upgrade neatlogs google-adk`
   - uv: `uv add --upgrade neatlogs google-adk`
   - poetry: `poetry add neatlogs@latest google-adk`
3. Background install; continue.
